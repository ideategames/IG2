Technology Credits

All of the games are written in html5 (CSS3 + javascript with the <canvas> context) using Phaser, an open source html5 game framework.
The web server is open source Apache with open source web2py as the app server and database.

Content Credits

Images used in these games are currently pulled from the web, and credits both in general and with specific referral URLs are presented in the game. As permission is received or the principle of 'fair use' is decided, chunks of content will be made available to public versions of the games.

For any copyright issues, please contact us immediately at ideategames@gmail.com. We have no intent to infringe copyright.
