Mastery Recognition (Buckets, Strings, Clues, and Collections)

If you achieve over 90% of the maximum score in 3 consecutive plays of the same subject in the same game, we grant you mastery status, and give you the option to post that achievement to your Facebook Activity Log. We require 3 consecutive games because the content for each play is small and we need to ensure that you have mastered a significant portion of the content.

Because Collections only requires you to find 4 objects, we require perfect scores. A perfect score is one with the minimum number of steps (rooms entered).

NOTE: Because mastery status can be posted to Facebook (optional), we will ask you to login to Facebook (if your browser is not already connected to Facebook). We store no information from your Facebook profile. If you login to Facebook you do not have to post to your Facebook Activity Log. The default for posts will be not visible on your timeline, you can choose to make it visible or not.

URL Shortcuts

If you have a favorite game or want to preserve your settings (eg, topic, level, your player name), you can do that all with parameters in the URLs. Here are the base URLs for each game:
  • Buckets: http://www.ideategames.org/Buckets?topic=&alias=&level=1&partner=
  • Strings: http://www.ideategames.org/Strings?topic=&alias=&level=1&partner=
  • Clues: http://www.ideategames.org/Clues?topic=&alias=&level=1&partner=
  • Rooms: http://www.ideategames.org/Rooms?topic=&alias=&level=1&partner=
  • Collections: http://www.ideategames.org/Collections?topic=&alias=&level=1&partner=
  • Paths: http://www.ideategames.org/Paths?topic=&alias=&level=1&clevel=1&partner=
Each of those URLs has three optional parameters, and the Paths game has one additional:

Parameters

parametermeaningvalid values
topicWhich topic you want to play. If blank, you get the topic select page. If this parameter contains a topic, you go directly to the game with that topic loaded. If the topic does not exist, you will get a spinner that doesn't stop.Topics differ for each game, and partners have special topics. The general ones available are: Museum, Science, Literature, Art, Composers, Culture, Movies, Nature, Cities. Some of these are not valid for some games (eg, Cities doesn't work well for Rooms and Collections).
aliasThis is only used for leaderboards (Strings and Clues at this time), and only appears when you post to a leaderboard.any text string
levelHow much help is provided for the player. At this time, only Rooms and Paths have levels.1, 2, 3 (only Paths uses 3 levels; invalid levels will be converted to 1)
partnerThis is for special installations, private collections of topics, such as for a class or other group. Any group of people can have their own set of topics by special arrangement.Any text string, but only by prior arrangement with Ideate Games.
clevel (Paths only)How much strategy the computer uses in Paths.1, 2, 3 (invalid levels will be converted to 1)


It might be nicer if we just exposed the URLs and let you just save favorites, but we decided (mostly to avoid confusion because the path names do not always correspond to the game names) not to expose the full URLs in the address bar. But this also gives you a chance to see all the parameters.
